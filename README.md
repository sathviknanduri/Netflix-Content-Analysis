# Netflix Content Strategy Analysis 📺🎬

## 📌 Project Overview
This project explores the Netflix dataset to understand content production trends, genre distributions, and geographical reach. The goal is to provide data-driven recommendations for Netflix's future content focus.

## 📊 Key Findings & Actionable Insights
* **Content Shift:** While movies dominate the catalog (~70%), TV show production has grown sharply since 2016, indicating a shift toward episodic engagement.
* **Target Audience:** The majority of content is rated **TV-MA** or **TV-14**, showing Netflix’s focus on adult and young adult demographics.
* **Production Hubs:** The US, India, and the UK remain the top producers, but there is a significant expansion in localized content from Asian markets.
* **Optimal Launch Timing:** Data suggests that the second half of the year (July–December) sees the highest volume of title additions.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib, Seaborn
* **Environment:** Google Colab

## 📈 Visualizations
This analysis includes:
* Univariate analysis of ratings and content types.
* Bivariate analysis comparing Movie vs. TV Show growth over the last 30 years.
